# Installation

Install the library using yarn or npm:

- Install with npm

```
npm i ngx-reactive-loading
```

- Install with yarn

```
yarn add ngx-reactive-loading
```

## Versions

| ngx-reactive-loading | Angular   | RxJS    |
| -------------------- | --------- | ------- |
| \>1.3.0              | \>=12.0.0 | \>6.5.3 |
